## Project Summary

### Projects/SQL-mini/
Mini project on retail transaction data using Python, SQLite, and SQL.

Contents:
- `Online Retail.xlsx` : source dataset
- `load_and_clean_data.py` : loads and preprocesses the raw Excel data
- `clean_and_save_db.py` : stores cleaned data in SQLite
- `run_sql_analysis.py` : runs SQL queries for basic sales analysis
- `country_sales_summary.csv` : exported query result
- `ReadMe.md` : project-level description
